What are Job Stories?

Last Updated: 6/27/2018

When _____ , I want to _____ , so I can _____ .

This is the format of a Job story. The goal is to break down user stories that are tied to specific user personas to just focus on a set of discreet "jobs".

One of the main arguments for this is that personas are hard. They're hard to create, maintain, and really makes a lot of assumptions.

Job stories focus on just the main function of the product.

When [condition] + What [action] = What [result]

Example Time!

When I go to sleep, I want to set an alarm, so that I can wake up in the morning.

If we try to use a user story = As a person who needs to wake up in the morning at a specific time and often oversleeps (?) I want to set an alarm, so that I can wake up in the morning.

Replace persona with situation. Because situations are universal, anyone can be in that situation.

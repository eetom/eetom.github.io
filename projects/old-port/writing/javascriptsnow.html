Learning Javascript

JavaScript snow effect on your website

Last Updated: 1/04/2018

I recently added some Javascript snowflake effect for this holiday season just to add a littel something festive to my website. I've seen it done a few different ways on different websites but wanted something light and simple. The tutorial I followed to create this effec is at Kirupa.com. Check out the screenshot below to see how it looks with the effect on.

Most of the steps you need will be on the tutorial website, but here are a few tips that I have found to make it a bit easier .

Step 1: Add all the assets

For this method of adding JS snowflakes, you'll need the CSS and JS file from the tutorial site.

Here is the CSS file ..

/*SNOW EFFECT STYLES*/
#snowflakeContainer {
position: absolute;
left: 0px;
top: 0px;
}
.snowflake {
padding-left: 15px;
font-family: Cambria, Georgia, serif;
font-size: 14px;
line-height: 24px;
position: fixed;
color: #FFFFFF;
user-select: none;
z-index: 1000;
}
.snowflake:hover {
cursor: default;
}

To make changes to the visuals of your snowflake, adjust the color to change the color of the snowflake.

Now downaload a copy of the Javascript file..

It's a legthy file, so we wont show the entire code here, but we can go over a few of the highlights. The file was created by Kuripa and is very well annotated and explains how each step works.

One of the more fun variable to change is the numberOfSnowflakes, it - as the name describes - determine the number of snowflakes on the screen at one time. I've got mine set to 100.

// Specify the number of snowflakes you want visible
var numberOfSnowflakes = 100;

Another cool section to look at is the controls for the speed and radius of the snowflakes. Each snowflake is randomly generated and have different sizes and speeds. The code uses the Math.random() function to randomize each snowflake. For example the radius of the snowflake is determined by 4+Math.random()*10 this allows for a number between 1 and 0 that is then multiplied by 10 and then add 4, for a range of 4 to 14.

var speed = 5+Math.random()*40;
var radius = 4+Math.random()*10;

Step 2: Connecting it to your website

Now that we have both the CSS and JS file we can get it to work with your website. To do this we'll need to add it to your website's HTML code.

First let's add the CSS file, I have the CSS file saved in my site's CSS folder, then I add the file by linking it in the HTML page <head> section I want the snowflakes to show up in..

<head>

<link href="your-path-to-the-css-file/snow.css" rel="stylesheet">
other code ..

Then the next step is to add the JS file to the HTML page..

other code ..

<script src="your-path-to-the-js-file/fallingsnow_v6.js"></script>
</body>

I tend to add JS script files at the bottom of my html so it doesn't slow down the rest of the HTML content from loading first

Step 3: Add a <div> for the snowflake

We're almost finished! If you load the page now.. no snowflakes happen. Why is that? It's because we need to have a element for the javascript to add and remove the snowflakes from. To do this we'll need to create a new <div> before the <body> tag in your HTML page.

<body>

<div id="snowflakeContainer">
<p class="snowflake">*</p>
</div>

... the rest of your code

This creates a <div> that is locked to the top of your browser page and covers the entire screen. This is where the JS file will put new snowflakes on top of the rest of the content. Now that you have all of it combined refresh the site and you should see some wonderful snow effect!

And that's it, all you need to add some holiday cheer to winterize your webpage. The key things to note are ..

  1. Add in the asset files - CSS and JS
  2. Link it in the HTML so it can be loaded
  3. Add the HTML code it needs to show up on the page

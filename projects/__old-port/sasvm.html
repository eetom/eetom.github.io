SASVM PLANT SAFETY SOFTWARE

SASVM is a Shell Canada project that was created to automatically check for safety conditions and maintence scheduling for the Shell Albian Sands Mines in Northern Alberta. The goal of the program was that it had to be fast to glance information and very low chance of making mistakes. All we had to start were CAD drawings of the plant that we had to render and then build a colour code system to match their conditional reports.

I started by creating all the assests used in the web app from the vector map to the 3D pipe diagrams. Then I worked with the engineers at the plant to develope a system that provided all the functions they would need for the job.

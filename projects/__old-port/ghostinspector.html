Ghost Inspector - Front-End Automated Testing



Ghost Inspector web app

Ghost Inspector is an automated front-end testing and monitoring web app that makes it super simple to create and maintain webs tests. I got interested in Ghost Inspector beacause I thought it would be super useful for designers to be able to build simple QA tests for their web projects.

My primary job was to work with with the team to redesign the entire Ghost Inspector experience, from a brand new marketing site, to UX improvements to the app, and even help with overall branding.

I worked closely with the backend developer to build the front end, and with managers to define and scope features, and even participated in customer support tickets to get a better understanding of how our customers were using the app.

Ghost Inspector Marketing Website

The original Ghost Inspector website was built on a simple wordpress site. My goal when I started to rebuild the site was to match the Runscope design cues but make Ghost Inspector it's own branding.

The site is built on the Jekyll platform and allows for fast system set up, easy test deployment, and ability to use components for common UI elements.

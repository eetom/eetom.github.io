Security rule builder

Cloudflare Firewall



Firewall and security app

Cloudflare’s firewall product is due for a major overhaul, it is one of Cloudflare’s earliest product and hasn’t had much changes since it first launched years ago. Over time new functions have been added on but without design oversight on improving user experience. The goal of this project was to redesign the way users would manage their firewall, the challenge of this is that there is a lot of moving parts that have to be taken into consideration in the new design. One of the ideas was a universal rule system to manage all firewall functions.

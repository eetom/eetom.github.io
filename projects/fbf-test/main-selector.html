Testing FBF concept modal

We are going to use this as a way to user test creating Filter Based Firewall rules.

We have 2 different design to test, the first is what we're calling 'Basic' (IFTTT style), this means that they will be using a more wizard form and have to build each piece of the filter in.

The second is an 'Advanced' mode (DSL style), where the user will be able to enter in a DSL into a textarea.

One of the things we will want to test would be what the transition between the two modes could be like? Should we only use one of them? Will they be allowed to go back to Basic if they choose Advanced? Theses are the questions we will try to answer with our customers in user tests.
